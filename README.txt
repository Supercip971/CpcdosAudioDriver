///CPCDOSAUDIODRIVER REPOSITORY///

Bienvenue sur le dépôt de code source pour le pilote AudioDriver pour Cpcdos.

Le projet consiste tout simplement à développer un driver son pour Cpcdos et ainsi contribuer au noyau.

Ce code est accessible à tout le monde et chaque personne peut y contribuer.

Lorsque le développement sera terminer, le driver sera intégrer à Cpcdos.



J'espère que vous serez nombreux à contribuer au pilote. Chrapati.
